Bom, existem alguns tipos de lista, ul, ol, li, vamos trabalhar com todas então não se preocupe.

Vamos aprender as diferenças das listas?

Vamos começar com a ul, a lista ul significa unordered list.
A ul significa que essa lista é uma lista sem ordem, aí você irá me dizer: Como assim? uma lista sem ordem? mas uma lista não é feita para dar ordem, organizar as coisas?
Sim, listas são par isso, mas essa representa uma lista onde não existe um grau de importância para os itens da lista, sacou?
