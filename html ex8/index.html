Você sabe deixar palavras em destaque?

Você já se viu em uma situação onde estava escrevendo um texto e precisou dar destaque em uma palavra?
Bom, em sites não é diferente e é por isso utilizaremos de certas TAG'S.

Negrito

Sabe quando temos aquela palavra que queremos transmitir algo forte.
E é por isso que temos a TAG "strong" ela consegue deixar a palavra, trecho ou oque você precisar em DESTAQUE

Itálico

O mesmo serve para itálico, sabe, aquela que deixa o texto caidinho(tipo eu por personagens com cabelo roxo)
Assim ó esse trecho aqui ta em itálico po e para fazermos isso utilizamos da TAG "em".

Outras TAG'S que podemos utilizar

Outras TAG'S legais que podemos utilizar:
"hr"= Adiciona uma linha, igual aquela ali no topo da página.
"Mark"= Marca um texto.
"del"= Risca um texto.
"ins"= Sublinha um texto
"code"= Formata um texto, utilizado para formatar códigos.
"Pre"= Podemos utilizar junto da TAG anterior, para pré formatar o código ou texto selecionado.
"Ol"/ul/li"= Tipos de listas, ol= Lista ordenada, ul=Lista não ordenada, li= Lista de item.
"Dl"= Lista de descrição.
"Dt"= Indica um termo.
"Dd"= descrição do termo.
"A"= Adicionar links.
"Address"= Indicar um endereço.
"small"= Transforma uma palvra, frase ou texto, o deixando pequeno.
"Sub/Sup"= Número pequeno.
"Abbr"= Significado de siglas.
"picture"= Adicionar imagens.
"img"= Junto com a TAG anterior, serve para procurar imagens.
" Q/blockquote "= "q" para citações simples/curtas, "blockquote" para uma citação completa/longa.
